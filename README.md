# Samuel Müller Portfolio

Premium 3D portfolio website for Samuel Müller, built with Next.js, React, TypeScript, Tailwind CSS, Framer Motion, Lenis and React Three Fiber.

## Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Check

```bash
npm run build
npm run lint
```

## Structure

```text
src/
  app/
  components/
  data/
  lib/
  styles/
public/
```

## Content

- 3D hero scene with animated laptop, code cube, rings and particles
- About section with personal story and professional tone
- Skills cards for frontend, backend/tools and creative work
- Project cards for Jarvis V3, Reiseagentur Webseite, CSR/FiveM, NexPlay, Python Experiments and this portfolio
- Timeline for 2025, 2026 and the 2027 EFZ goal
- Contact buttons for email and GitHub
